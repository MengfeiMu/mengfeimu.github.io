通过邮箱发送到kindle上

绑定到kindle上的邮箱可以直接发送到该接收邮箱，接收邮箱的地址在这里查看：

kindle -> 全部设置 ->您的账户 -> （最后一行）xxxx#kindle.cn。

如果要用其他邮箱发送到该接收邮箱，则需要添加：

网页版amazon.cn -> 管理我的内容和设备 -> 个人文档设置 -> 已认可的发件人电子邮箱列表 -> 添加认可的电子邮箱

Kindle个人文档服务目前只能转换并发送以下类型的文档：
Microsoft Word (.doc, .docx)
RTF文件(.rtf)
HTML(.htm, .html)
TXT文件(.txt)
Zip, x-zip压缩文件
Mobi格式电子书

JPEG (.jpg), GIF (.gif), Bitmap (.bmp) 和 PNG (.png) 格式的图片。
Adobe PDF (.pdf)文档可维持原文件格式，发送至您的Kindle。

Adobe PDF (.pdf)文档可转换为Kindle格式发送，目前处于试用阶段。

AZW和AZW3格式无法用此方法发送。

Windows电脑上同步

通过USB把kindle连接到电脑，把下载的mobi文件复制粘贴到如下路径：

kindle/documents/downloads

支持的书籍格式

kindle支持mobi、azw 或 azw3格式，不支持epub格式（ibooks支持）。最好不要在kindle上读PDF，一般效果不太好。
