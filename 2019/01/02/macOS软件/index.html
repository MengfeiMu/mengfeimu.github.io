Alfred

免费搜索工具，购买powerpack后有更高级的功能，如剪贴板记录、workflow等。

magnet

移动窗口自动排列。

iThoughtsX

画思维导图软件。

Sublime Text 3

详情请见：https://mengfeimu.github.io/2018/10/09/tool/2018-07-27-sublime/

Typora

免费Markdown编辑器，可输出各种格式，如pdf、word等。

iTerm

终端工具，代替自带的终端。

Oh my zsh

更好用的终端配置，如增强版的自动补全功能。

TeamViewer

屏幕共享与远程登录软件。免费。

Zotero

文献管理软件。

iCHM

免费的chm文件阅读器。

Paragon NTFS for mac

一般NTFS格式的移动硬盘在macOS下只可读不可写。这款软件可以让希捷移动硬盘在macOS下可写。

Github Desktop

Lightroom Classic CC

照片调色与修图软件。

XLD

音乐文件格式转换工具。无损转换为iTunes AAC格式（后缀名.m4a），转换完成后自动添加到iTunes。

输出格式选择：MPEG-4 AAC.

勾选：Add encoded files to iTunes if possible, Library

IINA

视频播放软件。

Downie

视频下载软件，复制粘贴或拖拽链接到软件即可，也可用软件内置的搜索功能直接搜索。试用下载YouTube视频很快。

Permute

格式转换工具，包括视频，音频，图片，甚至PDF文件。
